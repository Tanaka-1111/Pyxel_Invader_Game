# 👾 Pyxel Invader Game

Pythonのレトロゲームエンジン「Pyxel」を使用して開発した、王道スタイルのインベーダーゲームです。
オブジェクト指向（クラス設計）を取り入れ、保守性と拡張性の高いコード作りにこだわりました。

## 🌟 作品概要
全3ステージ構成のシューティングゲームです。
ステージが進むごとに敵の数や配置が変わり、最終ステージには体力ゲージ付きのボスが登場します。

* **ステージ1**: 雑魚敵 3体（横一列）
* **ステージ2**: 雑魚敵 8体（2列の波状攻撃）
* **ステージ3**: ボス戦 1体（3方向弾と耐久力持ち）

## 🎮 操作方法
* **左右矢印キー (`←` `→`)**: プレイヤーの移動（画面外にはみ出ない自動補正あり）
* **スペースキー (`SPACE`)**: 弾の発射（画面内に最大5発まで連射可能）
* **`R` キー**: タイトル画面からのスタート、およびゲームオーバー・クリア時のリスタート

* Python 3.x がインストールされていること
